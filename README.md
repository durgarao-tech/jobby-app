# Jobby App

A job search application built using React.js with authentication and API integration.

## Features
- User Login Authentication (JWT)
- Protected Routes
- Job Listings & Job Details
- Search and Filter Jobs
- Responsive Design

## Tech Stack
- React.js
- JavaScript
- CSS
- REST API
- JWT Authentication
